Summary: Wind Speed vs. Latitude

The motivation for this analysis was to see how local weather conditions are affected by position relative to the equator. Data from over 500 cities randomly located around the globe are being used along with the data from OpenWeatherMap API.

The data used includes temperatures, cloudiness, wind speed and humidity. This site puts that all together along with the source data for an easy overview of the relationships.
